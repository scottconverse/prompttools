prompttools

Git-native CI tooling for prompt engineering.

The repo-native prompt DevEx layer for AI product teams, OSS maintainers, and startups that keep prompts in version control.

Tools

Seven focused packages that work standalone or together.

promptlint

Static analysis for prompt files. Catches ambiguity, missing placeholders, excessive length, and more.

34 rules auto-fix

promptfmt

Auto-formatter that normalizes whitespace, heading structure, placeholder style, and section ordering.

5-rule pipeline watch mode

prompttest

Test framework for prompt files. Write assertions against prompt structure, content, and behavior.

15 assertion types JUnit XML

promptcost

Cost estimator with model comparison tables and budget enforcement for CI pipelines.

model comparison budget enforcement

promptdiff

Semantic diff for prompt changes. Shows message-level diffs, variable impact, token deltas, and breaking change detection.

breaking changes CI gate

promptvault

Version control and registry for prompt assets. Semantic versioning, dependency resolution, lockfiles, and a searchable catalog.

semver lockfiles

prompttools-core

Shared foundation providing prompt parsing, tokenization, and model profiles used by all tools.

parsing tokenization profiles

Quick Start

Install from PyPI and start using each tool right away.

Install

pip install promptlint-ai promptfmt-ai prompttest-ai promptcost-ai promptdiff-ai promptvault-ai

promptlint

# Lint all prompt files in a directory
promptlint check ./prompts/

# Auto-fix issues
promptlint check --fix ./prompts/

promptfmt

# Format prompt files
promptfmt format ./prompts/

# Check without modifying
promptfmt format --check ./prompts/

prompttest

# Run tests
prompttest run tests/

# Output JUnit XML for CI
prompttest run tests/ --format junit

promptcost

# Estimate costs across models
promptcost estimate prompt.yaml --compare

# Enforce a budget ceiling in CI
promptcost budget prompt.yaml --limit 0.05

promptdiff

# Compare two prompt files
promptdiff old.yaml new.yaml

# CI gate: exit 1 on breaking changes
promptdiff old.yaml new.yaml --exit-on-breaking

promptvault

# Initialize a prompt package
promptvault init --name @my-org/my-prompts

# Publish and install
promptvault publish
promptvault install
